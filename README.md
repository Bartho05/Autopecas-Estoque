# Sistema de Estoque de Autopeças

Sistema de gerenciamento de estoque para lojas de peças automotivas.

## Tecnologias

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- HTML
- Bootstrap
- JavaScript

## Funcionalidades

- Cadastro de produtos
- Listagem de estoque
- Atualização de produtos
- Remoção de produtos

## Como rodar o projeto

1. Iniciar MySQL no XAMPP
2. Criar banco:

CREATE DATABASE autopecas;

3. Rodar aplicação

AutopecasEstoqueApplication

Acesse:

http://localhost:8080
